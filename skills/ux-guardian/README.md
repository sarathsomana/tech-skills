# UX Guardian 🎨

`ux-guardian` performs systematic aesthetic audits, CSS refactoring, and animation polishing to elevate UI quality to a premium standard. It enforces human visual approval for all subjective design changes.

## Overview

Good UX isn't accidental — it requires consistent design tokens, accessible color palettes, responsive layouts, and thoughtful micro-animations. This skill audits existing UIs for aesthetic debt and systematically fixes it, but always defers to human judgment for subjective decisions.

## Execution Phases

| Phase | Persona | Goal |
| :--- | :--- | :--- |
| **SCOUT** | The Connoisseur | Audit existing UI for inconsistencies, accessibility issues, and design debt. |
| **STRATEGIZE** | The Design Director | Create a prioritized CSS refactoring and animation enhancement plan. |
| **REFINE** | The Artisan | Apply design token standardization, CSS fixes, and micro-animations. |
| **WAIT_FOR_HUMAN** | The Collaborator | **Mandatory** visual confirmation before proceeding — the agent cannot self-approve design. |
| **CRITIC** | The Design Critic | Final quality gate for design consistency and accessibility compliance. |

## Core Directives

- **Visual Confirmation is Mandatory**: The agent never self-approves UI changes. Every change goes through `Wait-For-Human`.
- **Premium Aesthetic Standard**: Targets modern design (glassmorphism, smooth gradients, micro-animations).
- **Accessibility First**: WCAG AA compliance is non-negotiable.

## Usage

Trigger this skill when your UI needs a design refresh, CSS cleanup, or accessibility audit. Provide the target UI files and any brand guidelines.
