# Roadmap

The detailed roadmap in the repo is [docs/ROADMAP.md](https://github.com/sarathsomana/tech-skills/blob/main/docs/ROADMAP.md). Below is a concise snapshot.

## Goals

Build a broad library of skills that follow the **phased state machine** model: explicit phases, personas, and artifact gates.

## Completed themes (high level)

- **Pre-phase 1 — Architecture refinement**: Artifacts, Scout persona, stderr as artifact, loop breakers, human-in-the-loop, hooks, exit-code gating, single-file YAML+Markdown skills, statelessness.
- **Phase 1 — Foundation**: Skill Architect, Base State-Machine Runtime.
- **Phase 2 — Core engineering**: Implementation Engineer, Root-Cause Isolator, Legacy Modernizer, Graphify v2.
- **Phase 3 — Visual & UX**: UX Guardian, Component Wizard.
- **Phase 4 — Hardening**: Security Sentinel, Perf-Optimus, Test Engineer.
- **Phase 5 — Ecosystem**: Multi-Agent Orchestrator, Skill Registry.

All items in [docs/ROADMAP.md](https://github.com/sarathsomana/tech-skills/blob/main/docs/ROADMAP.md) are marked complete as of that document; future roadmap updates will appear there first.
